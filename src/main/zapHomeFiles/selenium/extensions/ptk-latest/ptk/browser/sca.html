SCA, or Software Composition Analysis, is an application security methodology that helps identify and manage risks associated with open-source software components used in a project. By scanning JavaScript dependencies, OWASP PTK will identify security risks associated with open-source and third-party components.
How to scan an application:
  • Click the "Scan in runtime" button and confirm it's starting
  • Start browsing the application you are scanning, e.g. click on different links, submit forms, etc.
  • Once completed open SCA scanner and stop the scan
  • Check the result
